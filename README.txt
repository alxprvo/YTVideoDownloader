YouTube video downloader using pytube.
Download a YouTube video using a YouTube link.

You need to install python and pytube


Max resolution is 720p.
Will download the video in "~/Videos/Youtube Downloads"


Use the command: python path/to/YTDownloader.py "https://www.youtube.com/full-yt-link"

For exemple:
python C:\Users\Mark\PythonProjects\YTDownloader\YTDownloader.py "https://www.youtube.com/watch?v=dQw4w9WgXcQ&ab_channel=RickAstley"
